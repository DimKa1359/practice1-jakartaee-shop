# Практическая работа №1: Интернет-магазин на JakartaEE

## Описание
Приложение с архитектурой JakartaEE: три слоя (данные, логика, представление).  
Использует EJB, JPA, Servlet + JSP, PostgreSQL, GlassFish.

## Требования
- Java 17+
- GlassFish 7
- PostgreSQL 12+

## Установка
1. Создайте БД `myshop` в PostgreSQL.
2. Выполните `sql/table.sql`.
3. В GlassFish создайте JDBC Pool и Resource.
4. Разверните `shop-app.war`.
